# Percival Mobile

[![Version](https://img.shields.io/badge/version-1.1.0-blue)]()
[![Platform](https://img.shields.io/badge/platform-iOS%20%7C%20Android-lightgrey)]()
[![Status](https://img.shields.io/badge/status-active-success)]()

A lightweight mobile userscript for **Granblue Fantasy** that automates raids and streamlines gameplay with a clean, mobile-friendly interface.

---

## ✨ Features

- ⚔️ **Auto Raid** – Automatically starts raids  
- 🤖 **Full Auto** – Enables auto combat on battle start  
- ⚡ **Quick Attack** – Instantly attacks when possible  
- 🔄 **Reload on Attack / Summon** – Faster clears via reloads  
- 📱 **Mobile UI Panel** – Simple floating controls  

---

## 📱 Requirements

You’ll need a userscript manager:

- **iOS:** Scriptable / Userscripts apps  
- **Android:** Kiwi Browser + Tampermonkey  

---

## 💻 Desktop Version

---

## ⚙️ Installation

1. Install a userscript manager  
2. Create a new script  
3. Paste this script into it  
4. Save and enable  
5. Open **Granblue Fantasy**

---

## 🚀 Usage

After loading the game, a **“Percival” button** will appear in the bottom-right corner.

Tap it to open the control panel.

---

## 🎛 Controls

### 🔘 Toggles

- **Auto Raid** – Starts raids automatically  
- **Full Auto** – Enables auto combat  
- **Quick Attack** – Uses attack immediately  
- **Reload on Attack** – Reloads after attacking  
- **Reload on Summon** – Reloads after summoning  

---

### ⏸ Pause / Resume

- **Pause** – Stops all automation  
- **Resume** – Re-enables instantly  

---

## 🧠 Tips

- Combine **Full Auto + Reload** for hands-free farming  
- Use **Quick Attack** for faster manual-style clears  
- Toggle features individually if something behaves oddly  

---

## ⚠️ Notes

- Automates gameplay actions — use at your own discretion  
- Game updates may break functionality  
- Built primarily for mobile browsers  

---

## 🛠 Troubleshooting

**UI not showing**
- Refresh the page  
- Ensure the script is enabled  
- Confirm correct URL  

**Automation not working**
- Toggle features off/on  
- Reload the page
